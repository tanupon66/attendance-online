# Attendance Online PWA v2

เวอร์ชันนี้เพิ่มระบบตามรายการ:
- รีเฟรชแล้วไม่หลุด login ด้วย localStorage session
- พนักงานรายชั่วโมง / รายวัน / รายเดือน
- รอบจ่ายรายเดือน / ราย 14 วัน
- สวัสดิการตามวันทำงาน หรือ Fix รายเดือน
- ปฏิทินบริษัท
- วันลา + อนุมัติ
- OT request + อนุมัติเท่านั้นถึงคิดเงิน
- กะงาน / shift
- ประวัติการทำงานของ user
- Payroll Slip สำหรับพิมพ์
- Export CSV รายละเอียดมากขึ้น
- ลบข้อมูลเวลาเข้าออก และเคลียร์ audit log
- แสดงเวลาเข้าออก 1 คน 1 รายการต่อวัน
- UI ใหม่แบบ responsive

## v2.4.0

ปรับปรุงตามการใช้งานจริง:

- เพิ่ม Payroll Slip Rule
  - พนักงานรายวัน/รายชั่วโมงพิมพ์ slip รายวันได้หลังมีเวลาออกงาน
  - พนักงานรายเดือนเห็นรายละเอียดได้ แต่ปิดการพิมพ์ slip ตามนโยบาย
- เพิ่มตั้งค่าวันเงินออก
  - วันเงินออกรายเดือน
  - วันเริ่มงวดราย 14 วัน
- เพิ่มปฏิทินแบบตารางเดือนจริง
  - แอดมินเห็นในแท็บปฏิทิน
  - พนักงานเห็นปฏิทินทำงานของตัวเอง
  - แสดงวันทำงาน วันหยุด กิจกรรม และวันเงินออก
- ปรับ CSV ให้ละเอียดขึ้น
  - Attendance CSV มีทั้ง DAILY_SUMMARY และ RAW
  - Payroll CSV มีข้อมูลรอบจ่าย วันเงินออก ค่าแรง สวัสดิการ OT หักสาย ยอดสุทธิ
  - Slip CSV แยกบรรทัด summary/day/benefit
- ปรับการคำนวณ Payroll ให้เก็บ dailyDetails และ benefitLines เพื่อใช้พิมพ์ slip/Export

การอัปเดต: อัปโหลดไฟล์ทั้งหมดใน ZIP นี้ทับในโฟลเดอร์ `v2/` แล้วเปิด `https://tanupon66.github.io/attendance-online/v2/?v=22`
