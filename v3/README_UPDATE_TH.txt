# Attendance v3.2.4 - Daily Summary Clear Update

แพตช์นี้เพิ่มระบบเคลียร์ข้อมูลในหน้า “สรุปสถานะรายวัน”

## ไฟล์ที่ต้องอัปเดต
- src/modules/summary.js
- service-worker.js

## ฟีเจอร์ใหม่
- เพิ่มกล่อง “เคลียร์ข้อมูลสรุปรายวัน” ในหน้า “สรุปสถานะรายวัน” เฉพาะแอดมิน
- เลือกวันที่เริ่มต้น/สิ้นสุดได้
- ต้องพิมพ์ CLEAR เพื่อยืนยัน
- มีหน้าต่าง confirm ก่อนลบจริง
- ลบเฉพาะ collection `attendanceSummary` เท่านั้น
- ไม่ลบรายการลงเวลาใน `attendance`
- บันทึก audit log ด้วย action `CLEAR_DAILY_SUMMARY`

## วิธีใช้
1. แตก zip นี้
2. วางไฟล์ทับ path เดิมในโปรเจกต์
3. Commit/Push ขึ้น GitHub Pages
4. เปิดหน้าเว็บแล้ว refresh หรือ clear cache หากยังเห็นเวอร์ชันเก่า

## หมายเหตุ
หลังเคลียร์สรุปแล้ว ให้เลือกช่วงวันที่เดิมและกด “คำนวณ/สร้างสรุป” ใหม่ เพื่อให้ Payroll ใช้ข้อมูลล่าสุด
