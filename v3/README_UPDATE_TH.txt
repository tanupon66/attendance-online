Attendance v3.2.7 Payday Calendar + Mobile Notification Update
==============================================================

ไฟล์นี้เป็นแพตช์เฉพาะไฟล์ที่ต้องอัปเดต ให้นำไฟล์ใน zip นี้ไปวางทับโปรเจกต์เดิมตาม path เดิม แล้ว commit/push ขึ้น GitHub Pages

ฟีเจอร์ที่เพิ่ม
---------------
1) หน้า “ตำแหน่งบริษัท”
   - เพิ่มหัวข้อ “วันจ่ายเงินพนักงานรายเดือน”
   - เปิด/ปิดวันจ่ายเงินรายเดือนได้
   - กำหนดได้ว่าจ่ายทุกวันที่เท่าไหร่ของเดือน 1-31
   - ถ้าเดือนนั้นไม่มีวันที่ที่กำหนด เช่น ตั้งวันที่ 31 แต่เดือนกุมภาพันธ์มี 28/29 วัน ระบบจะใช้วันสุดท้ายของเดือนแทน
   - กำหนดหัวข้อในปฏิทินได้
   - กำหนดแจ้งเตือนล่วงหน้า 0-7 วันได้
   - มีปุ่มทดสอบแจ้งเตือนบนเครื่องนี้

2) หน้า “ปฏิทินบริษัท”
   - ดึงวันจ่ายเงินรายเดือนจาก settings/company มาแสดงอัตโนมัติ
   - วันจ่ายเงินจะแสดงเป็น badge “💰” บนวันที่นั้น
   - ถ้าวันจ่ายเงินตรงกับวันทำงาน วันนั้นยังเป็น workday ตามปกติ ไม่ถูกเปลี่ยนเป็นวันหยุด
   - วันจ่ายเงินเป็นเพียงป้ายแจ้งเตือน/ข้อมูลประกอบในปฏิทิน
   - ไม่เขียนทับ calendarEvents เดิม จึงไม่ทำให้ข้อมูลวันทำงาน/วันหยุดเดิมเสีย

3) แจ้งเตือนบนโทรศัพท์ / PWA Notification
   - เมื่อพนักงานรายเดือนหรือแอดมินเปิดระบบ และถึงช่วงแจ้งเตือนวันจ่ายเงิน ระบบจะสร้าง notification ในศูนย์แจ้งเตือน
   - ถ้าเครื่องนั้นอนุญาต Notification แล้ว ระบบจะแสดงแจ้งเตือนบนโทรศัพท์ด้วย
   - เพื่อเปิดสิทธิ์ ให้เข้าเมนู Notification Center แล้วกด “เปิดแจ้งเตือนบนเครื่องนี้” หรือกด “ทดสอบแจ้งเตือนบนเครื่องนี้” ในหน้าตำแหน่งบริษัท

หมายเหตุสำคัญ
-------------
- ระบบนี้เป็น PWA บน GitHub Pages ไม่มี backend server สำหรับ push แจ้งเตือนตอนปิดแอปสนิทแบบ 100%
- แจ้งเตือนมือถือจะทำงานเมื่อผู้ใช้เคยอนุญาต Notification และมีการเปิด/เข้าใช้งานแอปในช่วงแจ้งเตือน
- ถ้าต้องการ push แจ้งเตือนแม้ปิดแอปสนิท ต้องมี Cloud Functions หรือ server ส่ง FCM เพิ่มในอนาคต

หลังอัปเดต

1) เปิดเว็บแล้วกด reload ถ้ายังเห็นเวอร์ชันเก่าให้ล้าง cache หรือรอ service worker อัปเดต
2) เข้าเมนู “ตำแหน่งบริษัท”
3) เปิด “วันจ่ายเงินพนักงานรายเดือน” และตั้งวันที่จ่ายเงิน
4) เข้าเมนู “ปฏิทินบริษัท” เพื่อดู badge วันจ่ายเงิน
5) เข้าเมนู Notification Center หรือกดทดสอบแจ้งเตือน เพื่ออนุญาต Notification บนมือถือ
