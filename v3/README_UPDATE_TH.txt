# Attendance v3.2.6 Payroll Deduction Update

แพตช์นี้เพิ่มระบบ "รายการหักเงิน" ในหน้า Payroll

## ไฟล์ที่อัปเดต
- src/modules/payroll.js
- service-worker.js
- README_UPDATE_TH.txt

## ฟีเจอร์ที่เพิ่ม
1. แอดมินเพิ่มรายการหักเงินได้ในหน้า Payroll
   - เลือกพนักงาน
   - เลือกวันที่หักเงิน
   - กรอกจำนวนเงินที่ต้องหัก
   - กรอกเหตุผลที่หักเงิน

2. Payroll จะนำรายการหักเงินไปคิดอัตโนมัติ
   - ดึงข้อมูลจาก collection: payrollDeductions
   - นำเฉพาะรายการที่ active !== false มาคิด
   - คิดตามช่วงวันที่ที่เลือกใน Payroll
   - รวมเข้ากับรายการหักเดิม เช่น มาสาย / ขาดงาน / ลาไม่จ่ายเงิน

3. Slip แสดงรายละเอียดรายการหักเงิน
   - วันที่หัก
   - จำนวนเงิน
   - เหตุผลที่หัก

4. Export CSV เพิ่มข้อมูลรายการหักเงิน
   - manualDeductionCount
   - manualDeduction
   - totalDeduction

5. แอดมินลบ/ยกเลิกรายการหักเงินได้
   - ระบบไม่ได้ลบ document ทิ้งทันที
   - แต่เปลี่ยน active เป็น false
   - เพื่อให้ยังตรวจสอบย้อนหลังได้

6. เพิ่ม audit log
   - CREATE_PAYROLL_DEDUCTION
   - DELETE_PAYROLL_DEDUCTION

## วิธีติดตั้ง
1. แตก zip นี้
2. วางไฟล์ทับ path เดิมในโปรเจกต์
3. commit และ push ขึ้น GitHub
4. เปิดเว็บแล้ว refresh แบบ hard refresh หรือ Clear site data ถ้า service worker ยังจำไฟล์เก่า

## วิธีใช้งาน
1. เข้าเมนู Payroll
2. เลือกช่วงวันที่งวดเงินเดือน
3. ที่กล่อง "รายการหักเงิน" ให้เลือกพนักงาน กรอกวันที่ จำนวนเงิน และเหตุผล
4. กด "บันทึกรายการหักเงิน"
5. กด "คำนวณ" หรือระบบจะคำนวณใหม่ให้อัตโนมัติหลังบันทึก
6. เปิด Slip จะเห็นรายการหักเงินแยกตามเหตุผล

## หมายเหตุ
รายการหักเงินจะถูกคิดตามวันที่ของรายการ หากวันที่อยู่นอกช่วง Payroll ที่เลือก จะยังไม่ถูกนำมาคิดในงวดนั้น
