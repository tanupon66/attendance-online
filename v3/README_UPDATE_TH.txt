# Attendance Online v3.2.5 - Payroll + OT Fix

แพตช์นี้แก้หน้า Payroll ให้ดึงข้อมูลถูกต้องขึ้นและนำ OT ที่อนุมัติแล้วมาคิดเงิน

## ไฟล์ที่อัปเดต
- `src/modules/payroll.js`
- `service-worker.js`
- `README_UPDATE_TH.txt`

## สิ่งที่แก้ไข
1. หน้า Payroll ไม่ได้อ่านจาก OT เดิม
   - เดิมดึงข้อมูลจาก `attendanceSummary` อย่างเดียว
   - ใหม่ดึงเพิ่มจาก `otRequests` เฉพาะรายการที่ `status = approved`

2. เพิ่มค่า OT เข้าเงินเดือน
   - สูตร: `ชั่วโมง OT × ค่าแรงต่อชั่วโมง × otMultiplier`
   - `otMultiplier` ใช้ค่าจากข้อมูลพนักงาน ถ้าไม่มีใช้ 1.5

3. เพิ่มปุ่มในหน้า Payroll
   - `สร้างสรุปรายวันใหม่ก่อนคำนวณ`
   - ใช้เมื่อมีการแก้เวลา/เพิ่มเวลาโดยแอดมิน/อนุมัตินอกรัศมี แล้วต้องการให้ Payroll ใช้ข้อมูลล่าสุดทันที

4. แสดงรายละเอียด OT ใน Payroll Card และ Slip
   - จำนวนชั่วโมง OT
   - ค่า OT
   - รายละเอียดรายการ OT ที่อนุมัติแล้ว

5. Export CSV เพิ่มคอลัมน์ OT
   - `approvedOtCount`
   - `approvedOtHours`
   - `otMultiplier`
   - `hourly`
   - `otPay`

## วิธีติดตั้ง
แตก zip แล้ววางไฟล์ทับ path เดิมในโปรเจกต์ จากนั้น commit/push ขึ้น GitHub Pages

## วิธีใช้งานหลังอัปเดต
1. ไปที่หน้า `Payroll`
2. เลือกช่วงวันที่
3. ถ้าเพิ่งมีการแก้เวลา/เพิ่มเวลา/อนุมัติรายการนอกรัศมี ให้กด `สร้างสรุปรายวันใหม่ก่อนคำนวณ`
4. กด `คำนวณ`
5. ตรวจว่า badge `OT` และ `ค่า OT` แสดงในแต่ละพนักงาน

## หมายเหตุ
Payroll ใช้ข้อมูล 2 แหล่งหลัก:
- `attendanceSummary` = วันทำงาน ชั่วโมงปกติ มาสาย ขาด ลา
- `otRequests` = OT ที่อนุมัติแล้วเท่านั้น

ดังนั้นถ้า OT ยังเป็น pending หรือ rejected จะไม่ถูกนำมาคิดเงิน
