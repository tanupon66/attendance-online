# Attendance Online v3.1 Full Stable Pack

เวอร์ชันเต็ม รวม Step 7 Full Pack + Patch ล่าสุด

## รวมฟีเจอร์
- Login + Anonymous Firebase Auth
- Employee Management
- Attendance: Selfie + GPS + Geofence
- Daily Summary: มาสาย / ขาดงาน / ข้อมูลไม่ครบ / วันลา / วันหยุด
- Company Calendar
- Leave Entitlements: กำหนดสิทธิ์วันลาแยกแต่ละคน
- Leave Request: เต็มวัน / ครึ่งวัน / รายชั่วโมง / ลาจ่ายเงิน / ลาไม่จ่ายเงิน
- Notification Center
- Profile + เปลี่ยน PIN
- Payroll + Payroll Slip
- Export Payroll CSV / Slip CSV / Summary CSV
- UI มือถือปรับ Bottom Menu ให้เลื่อนซ้ายขวาได้
